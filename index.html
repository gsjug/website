Abstract

What is a JSON Web Token (JWT) and why do you care? JWTs are a stateless, standardized way to represent user data.

This talk will discuss why JWTs matter and the nuts and bolts of JWTs. We'll also discuss how you might use a JWT in your web or other Java application and how to avoid common mistakes.
