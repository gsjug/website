September 11, 2012
Technical Overview of Dart

This month''s meeting will be a technical overview of the Dart programming language. Dart is a new class-based programming language for creating structured web applications. Developed with the goals of simplicity, efficiency, and scalability, the Dart language combines powerful new language features with familiar language constructs into clear, readable syntax. We will take a look at some sample code that is included in the Dart SDK and editor.
