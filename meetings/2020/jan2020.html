January 14, 2020
Getting Started with Quarkus

Quarkus, a Kubernetes Java native framework developed by Red Hat, was designed for GraalVM and OpenJDK HotSpot. Quarkus leverages technologies such as MicroProfile, Hibernate, Netty and RESTEasy.

This session will provide an overview of Quarkus followed by demos on how to get started.
