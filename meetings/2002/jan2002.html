January 2002 Meeting Information

January 8, 2002
Sorting in Java

This month's meeting is based on an article published in the December 2001 issue of Java Solutions, a bimonthly supplement to the C/C++ Users Journal. Sorting is relatively easy in Java until the need arises to sort user-defined objects containing multiple data. The Comparable and Comparator interfaces allows the developer to define what data can be sorted in a particular object. We will review examples from the article and an example created using data generated from a small Access database.

Meetings are held every second Tuesday of the month at the Scotch Plains Rescue Squad second floor meeting hall starting promptly at 7:30pm. Meetings are open to the public, and visitors are encouraged to join ACGNJ.
