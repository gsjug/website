Java Users Group

December 2002 Meeting Information

December 10, 2002
Technical Overview of Ant: A Java-Based Build Tool

This month's user group meeting will be presented by Paul Furbacher. Paul will be presenting a technical overview of Ant, a Java-based build tool that is available from The Apache Jakarta Project, the folks that also develop Tomcat. Also, there will be a review of small, interesting projects from Jakarta's set of projects called Commons.
