February 2002 Meeting Information

February 12, 2002
Sunrise/Sunset Calculator

This month's meeting is a source code review of a sunrise/sunset calculator for a specific date and location. The original source code is based on a sun clock applet located at http://www.merrymeet.com/minow/sunclock/SunClock.html, and modified to be a simple application that displays the sunrise/sunset for a single location on a particular day. I will attempt to make the application a bit more dynamic to at least display sunrise/sunset times for a list of cities around the world.
