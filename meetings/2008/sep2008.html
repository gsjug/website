September 2008 Meeting Information

September 9, 2008
Rapid Web Application Development with Grails

This month's presentation will demonstrate how to rapidly build a web application with Grails. Grails is an open source, MVC framework that runs on the JVM. You can use both Java and Groovy to develop your application. This presentation will feature building a web application from scratch.

Meetings are held every second Tuesday of the month at the Scotch Plains Rescue Squad second floor meeting hall starting promptly at 7:30pm. Meetings are open to the public, and visitors are encouraged to join ACGNJ.
