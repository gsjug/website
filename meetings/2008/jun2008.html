June 2008 Meeting Information

June 10, 2008
Using Java to Interact with the OS

Barry Burd will be making a presentation to describe the somersaults he has performed in his attempt to unlock an obsolete screen saver. Barry's tools include the following:
  • Java AWT Robot: Automates mouse actions and keystrokes in the native system
  • JNA (Java Native Access): A new tool for accessing native libraries (Windows DLLs and maybe even Paul F.'s favorite Macintosh libraries)
  • cmdow: An interesting Windows command line tool
  • various disassemblers, decompilers and flow-charting tools, and...
  • ... possibly other tools.
So far, Barry hasn't unlocked the screen saver. But who knows? It may happen during the meeting!

Meetings are held every second Tuesday of the month at the Scotch Plains Rescue Squad second floor meeting hall starting promptly at 7:30pm. Meetings are open to the public, and visitors are encouraged to join ACGNJ.
