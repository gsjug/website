April 27, 2021
Functional Reactive with Core JDK

We are happy to announce that Sven Ruppert will be our special guest for this month's meeting.

There are a lot of reactive frameworks in the field, some with functional languages for the JVM. However, Java gives you both. But how you could combine this and use this without adding the next big framework to your project?

We will have a Core Java journey to explore a lot of excellent possibilities based on the JDK.
