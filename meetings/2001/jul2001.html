July 2001 Meeting Information

July 10, 2001
A Comparison of Java and C++

• Random Access 7:30pm – 8:00pm
Open discussion on any computing related topic.
• Java Q&A and/or short Java API Presentation 8:00pm – 8:30pm
A comparison of Java and C++.
• Main Presentation 8:30pm – 10:00pm
A source code review of a small wind chill application written in Java and C++ to demonstrate some of the language differences.

Meetings are held every second Tuesday of the month at the Scotch Plains Rescue Squad second floor meeting hall starting promptly at 7:30pm. Meetings are open to the public, and visitors are encouraged to join ACGNJ.
