September 8, 2015
Technical Overview of Google Polymer — Silver Bullet & Silver Lining

This month's meeting will be presented by "Wild Bill" Brutzman. Polymer-Project.org is a cool new Google framework for web apps that...enables mortal developers to quickly and all-at-once...create slick professional user interfaces that auto-scale to smart phones, tablets, notebooks, and desktops. Polymer is easy-to-use, has its own drag-and-drop IDE, and has a robust set of built-in (expandable) tools that allow for things like CSS to be handled automatically...or manually if desired. The goal for this talk is to demo how Polymer can facilitate quickly whipping up a robust web app...connected to a cloud database. Because Polymer has many parts, attendees able to preview the YouTubes would benefit.

Bill Brutzman does IT at a manufacturing company in Bergen County, New Jersey.
