February 2024 Meeting Information

February 21, 2024
Introduction to Oracle's MERN Stack

We are happy to announce that Doug Drechsel will be our special guest for this month's meeting.

This talk will present a new Oracle stack for Mobile and Web developers based upon a NodeJS/Express server architecture and a new storage adapter for Oracle written with NodeJS libraries that runs against a JSON database.

Doug will quickly build a server locally using Parse Platform that will run against the new Free23c docker container. Once the server is up and running, he will demonstrate a mobile application written in Flutter that runs against the stack.
