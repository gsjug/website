March 2024 Meeting Information

March 12, 2024
Lightning Talks
void main()

We are happy to announce that Drew University students: Muhammad Mir; Jared Simonetti; and Joseph Holzman; will present 15-minute lightning talks, and GSJUG Co-Director, Chandra Guntur, will be present the main topic at this month's meeting!

Muhammad Mir will present “Abstract Syntax Trees”

Jared Simonetti will present “Game Development Using Löve 2D and Lua”

Joseph Holzman will present “New Jersey Resource Finder Mobile Application: The Importance of Human-Centric Tech Solutions”

Chandra Guntur will present “void main()”

There are many changes in the Java language aiming to reduce boilerplate and make the language more straightforward and attractive to new developers. This talk will cover recent features by live refactoring an application to the newest Java version. You will learn the benefits and applicability of new features while writing a more readable code.
