October 2024 Meeting Information

October 8, 2024
Using MySQL Document Store with Java

We are happy to announce that Scott Stroz will be our special guest for this month's meeting.

MySQL Document Store offers developers the best features of NoSQL and relational, SQL-based databases.

In this session, we will discuss how we can use Java to access a MySQL Document Store, create/remove collections, create/update/remove documents, retrieve, filter, and sort document lists, and leverage raw SQL to retrieve aggregate data for reporting purposes.
