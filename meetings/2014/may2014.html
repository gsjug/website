Technical Overview of Vert.x

This month's meeting will be a technical overview of Vert.x, a lightweight, high performance application platform for the JVM that's designed for modern mobile, web, and enterprise applications. Simple, powerful, APIs enable you to write non-blocking network enabled applications with ease. No complex configuration or boilerplate is required. Application components can be written in Java, JavaScript, CoffeeScript, Ruby, Python or Groovy. Several of these programming languages can be used in a single app. There are plenty of examples for review from the Examples section of the Vert.x web site.
