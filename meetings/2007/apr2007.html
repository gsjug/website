Java Users Group

April 2007 Meeting Information

April 10, 2007
Technical Overview of Java Code Quality Tools

Paul Furbacher will be looking at a number of code quality measurements and tools. Open source projects that analyze code such as FindBugs, PMD, and Checkstyle will be reviewed with a focus on setting them up with Ant. Finally, Paul will discuss the take home lesson from the warnings/errors these tools report.

Also, at least one (1) IntelliJ IDEA license will be raffled off at the end of the meeting!
