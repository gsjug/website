Developer Career Masterplan

We are happy to announce that Bruno Souza and Vinicius Senger will be our special guests for this month's meeting who will each present their own talk!

Developer Career Masterplan

Looking for a plan to take your career to senior levels and beyond, and help you reach an outstanding reputation and the best positions in the market?

From getting the first job to getting the best job, from technology to creating amazing impact in the world, this interactive career conversation will both offer a proven master plan, but Bruno will also discuss real career issues that you may have.

Based on the book of the same name, this presentation will walk you through the most important steps of technical career advancement, and will help you build your personal plan for an expert career.

Java & Cloud: How Games, Music, and AI can help you build Massive Enterprise Workloads in the Cloud

Combine your Java knowledge with the most requested Cloud skills in the market.

This interactive talk will be a tour around Java and Cloud technologies. From Generative AI to modern Java serverless backend using Quarkus, from Roblox Java Castle Game with Java backend, to IoT devices, and AI powered camera and sound recognition.

You will learn how Amazon and AWS are using the latest Java versions, combining all those technologies, to run massive cloud workloads.

Come see how Open Source Java is being used in the larges cloud installations in the planet, and learn how you can benefit from it in your career.
