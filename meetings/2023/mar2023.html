March 14, 2023
Designing for the Serverless Cloud

We are happy to announce that Cameron Purdy will be our special guest for this month's meeting.

What does "serverless" even mean? And why are "lambdas" the wrong answer? What would a language and a runtime inspired by Java look like if it were being designed today, for the serverless cloud?

In this talk, we'll dive into Ecstasy and the XVM, a language and runtime model designed for the serverless cloud, and built using the latest Java technologies and running on the JVM. We'll talk about fibers and concurrency, autonomic management, software containers with dependency injection, and our goal to reduce each application's carbon footprint.
