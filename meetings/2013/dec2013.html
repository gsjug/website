December 10, 2013
Technical Overview of Utilities, Maven & Github

This month, we will have a "working" meeting where we will focus on the utilities, Maven and Github. There seems to be a new way of downloading and installing open source tools using Maven. This was seen with open source tools such as Spring Data for MongoDB and Jongo. Github has been widely adopted as a collaboration tool for source code. Spring Data for Mongo DB could also be "forked" using Github. We will start by review the presentation given by Yakov Fain at his last Princeton Java Users Group meetup and see what we can build.

Meetings are held every second Tuesday of the month at the Scotch Plains Rescue Squad second floor meeting hall starting promptly at 7:30pm. Meetings are open to the public, and visitors are encouraged to join ACGNJ.
