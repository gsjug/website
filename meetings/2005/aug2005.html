August 2005 Meeting Information

August 9, 2005
Special Project

A summer break is usually scheduled for the month of August. However, there is interest to work on a special project for ACGNJ on our regularly scheduled meeting night. The plan is to develop a web front end for the ACGNJ Attendance Database. This database contains attendance data for all of the special interest group (SIG) meetings. If you are interested in participating in the development of this application, I would appreciate if you could arrive between 6:00-6:30pm. This will allow us to get an early start, get as much accomplished as possible, and still adjourn by 10:00pm. My laptop will be configured with the latest versions of the necessary open source tools such as:
  • MySQL
  • Hibernate
  • Cayenne
  • Spring
  • Tapestry
  • JSF
  • Tomcat
