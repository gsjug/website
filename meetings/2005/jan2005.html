Enhancing Tasks with AntFlow

This month's meeting will be an overview of AntFlow, a tool for the automation and scheduling of data system tasks, including those with complex dependencies and workflow logic. AntFlow builds upon Ant to simplify system automation using pipelines of hot folders to perform a given task. AntFlow enhances Ant with the following features:
  • hot folder triggers
  • powerful task scheduling
  • workflow and error handling
