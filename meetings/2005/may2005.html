May 10, 2005
Rapid GUI Development with Thinlet

This month we will review the Thinlet GUI toolkit which uses only a single Java class to parse the hierarchy and properties of the GUI. It handles user interaction, calls business logic, and separates the graphic presentation (described in an XML file) and the application methods (written as Java code). Its compressed size is 39KB, and it is LGPL licensed. Thinlet runs with Java™ 1.1 (IE's default JVM) to 1.4, Personal Java™, and Personal (Basis) Profile™. Swing isn't required. We will review some of the Thinlet demos and the application from the October 2002 meeting (Observer Design Pattern) that was recreated using Thinlet for comparison to the original.
