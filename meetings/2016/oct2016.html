Vaadin ‐ A Java Tool for Web Development

This month, our own “Wild Bill” Brutzman will once again present to our group...

Wikipedia says...

Vaadin is an open source web framework for rich Internet applications. In contrast to JavaScript libraries and browser-plugin based solutions, Vaadin features a server-side architecture, which means that the majority of the logic runs on the servers. Ajax technology is used at the browser-side to ensure a rich and interactive user experience. On the client-side Vaadin is built on top of and can be extended with Google Web Toolkit and Polymer.

Bill Brutzman says...

Wow! All of the heavy complexity is gone. Vaadin makes coding streamlined, understandable, fast and fun. With Vaadin, mere mortals can build robust apps in HTML, JavaScript, Java EE, Spring, Scala, and Groovy not getting lost.

So, maybe instead of Java-Sig, we'll start calling this group Vaadin-Sig.

Bill works as an IT Manager at a manufacturing company in Bergen County.
