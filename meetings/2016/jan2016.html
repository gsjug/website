January 12, 2016
Firebase 101

This month, “Wild Bill” Brutzman will present on Firebase. Firebase is a cool new cloud NoSQL database from Google. In FireBase, data is stored as JSON synched to all connected clients in real time, that is, mobile app, tablet, and desktop users do not have to hit the browser refresh button. FireBase Includes built-in user authentication to e-mail, Facebook, Twitter, GitHub, and Google with built-in SSL and hosting. Developers can code via JavaScript, Java, REST, Objective C and can get started with FireBase for free. FireBase code will be demonstrated with an example website using Data Structures, CRUD, UserAccess, Security, Latency, Presence, and Deployment.

Meetings are held every second Tuesday of the month at the Scotch Plains Rescue Squad second floor meeting hall starting promptly at 7:30pm. Meetings are open to the public, and visitors are encouraged to join ACGNJ.
