Java Users Group

November 2003 Meeting Information

November 11, 2003
Use of Java in Embedded Systems

Manny Goyenechea will be making a comprehensive presentation on the use of Java in embedded systems. There will be an introduction to embedded systems, challenges for developing applications on them, and how Java fits into the picture. Included will be discussions on why to use Java vs. other computer language, when to use Java and when not to use Java, and which embedded systems use Java. Manny will also demonstrate an application that collects and stores temperature information.
