June 10, 2003
Binding Java Objects to XML Data with JAXB

This month's meeting will be presented by Paul Furbacher and feature a presentation of Java and XML data binding with examples based on Sun's Java Architecture for XML Binding (JAXB). Java developers increasingly must handle data in the form of XML (e.g., Web Services). Parsers and Document Object Models (DOMs) are clumsy when the more natural expression of the XML data is a set of Java objects and collections. A number of data binding frameworks exist to assist the developer mediate the interchange of XML and Java objects. They offer code generation and, to some extent, customization.
