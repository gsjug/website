January 2019 Meeting Information

January 8, 2019
Invest in Your Java Katalogue

We are happy to announce that Don Raab will be our special guest for this month's meeting.

A Code Kata is a hands-on programming exercise which helps hone your skills through practice and is usually set up as a series of unit tests which fail. Your task is to make them pass. Don will explain the approach he has taken in developing Java Code Katas, and discuss best practices around them. He will walk through a number of examples, including Katas teaching features of Java 8, 9 and 10, Eclipse Collections, and the Java Time library.
