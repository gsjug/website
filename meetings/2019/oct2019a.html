October 8, 2019
Java on Azure: How Microsoft Learned to Love Java

We are happy to announce that Reza Rahman will be our special guest for this month's meeting.

This session is a fast-paced tour of all things Java and Azure at the modern Microsoft of today. Reza will first talk about the why, how and what of Java and Microsoft. He will then dive right into the broad range of tools, services and APIs that Microsoft offers around Azure for Java developers.

Reza will show a small but representative demo of a Java application most developers would feel familiar with running well on Azure. Lastly, he will discuss the roadmap for what Java developers can expect from Microsoft in the future. This session is as much about sharing what Microsoft offers today as it is about listening to what the Java community wants to see from Microsoft tomorrow.

Key Points
  • Why, how and what of Java and Microsoft
  • Tour of the broad range of tools, services and APIs that Microsoft offers around Azure for Java developers
  • Roadmap for what Java developers can expect from Microsoft in the future

Meetings are held every second Tuesday of the month at the Dorothy Young Center for the Arts, Room 106, at Drew University starting promptly at 6:30pm. Meetings are open to the public, and visitors are encouraged to join ACGNJ.
