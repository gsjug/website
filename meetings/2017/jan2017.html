Technical Overview of Speedment

Speedment, is a Java 8 toolkit and runtime for accelerated SQL database applications. A portmanteau blend of “speed” and “development,” Speedment accelerates development speed and makes database programming easy and fun! When Speedment Open Source is used for database querying, you do not have to learn a new API or use complex (object-relational mapping) ORMs. Everything is standard Java 8 and works out of the box. There will be a brief overview of Speedment followed by a demonstration on how to build a small database application in real time.
