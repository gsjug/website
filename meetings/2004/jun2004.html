June 8, 2004
Using Web Services with the J2EE 1.4 Platform - Part 1

Abigail Siner will be making a presentation on web services and J2EE 1.4. The J2EE 1.4 platform provides comprehensive support for web services through the JAX-RPC API, as well as JAXR, SAAJ, and more (there is definitely no shortage of acronyms here!). These APIs provide interoperability with web services based on standards such as Web Services Description Language (WSDL), Simple Object Access Protocol (SOAP), the WS-I Basic Profile, and Universal Description, Discovery and Integration (UDDI). This talk will discuss some of these standards, and then show how J2EE 1.4 allows people to develop web services with having to understand or directly use these standards (in theory).
