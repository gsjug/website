July 2004 Meeting Information

July 13, 2004
Using Web Services with the J2EE 1.4 Platform - Part 2

Abigail Siner will be continuing her presentation on web services and J2EE 1.4. The J2EE 1.4 platform provides comprehensive support for web services through the JAX-RPC API, as well as JAXR, SAAJ, and more (there is definitely no shortage of acronyms here!). These APIs provide interoperability with web services based on standards such as Web Services Description Language (WSDL), Simple Object Access Protocol (SOAP), the WS-I Basic Profile, and Universal Description, Discovery and Integration (UDDI). This talk will discuss some of these standards, and then show how J2EE 1.4 allows people to develop web services with having to understand or directly use these standards (in theory).

Meetings are held every second Tuesday of the month at the Scotch Plains Rescue Squad second floor meeting hall starting promptly at 7:30pm. Meetings are open to the public, and visitors are encouraged to join ACGNJ.
