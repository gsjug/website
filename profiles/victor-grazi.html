Victor Grazi

Victor Grazi is Senior Vice President at BNY Pershing X. He is a Java Champion, author of several open source projects including the popular Java Concurrent Animated, and co-author of the book, Real-World Java: Helping You Navigate the Java Ecosystem. He serves on the Java Community Process Executive Committee, won an Outstanding Spec Lead award for JSR 354: Money and Currency API, and has presented on Java-related topics across the globe.
