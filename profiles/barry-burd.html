Barry Burd

Barry has been a regular attendee and frequent contributor of the Java Users Group since May 2003. He is a Professor of Math and Computer Science at Drew University in Madison, New Jersey. Barry has written numerous books (including Java for Dummies and Java Programming for Android Developers For Dummies) and has presented at numerous venues including the annual Trenton Computer Festival and the annual Droidcon NYC Conference.

He was anointed Java Champion in November 2020. For more detailed information about Barry, please visit his personal web site, burd.org.
