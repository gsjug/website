What's New

JDK 23
  • Released on September 17, 2024
  • Java 23 Launch Stream Event

JDK 24
  • JEP 472: Prepare to Restrict the Use of JNI
  • JEP 475: Late Barrier Expansion for G1
  • JEP 484: Class-File API
  • JEP 489: Vector API (Ninth Incubator)
JDK 24 Release Schedule
  • Rampdown Phase One (fork from main line): December 5, 2024
  • Rampdown Phase Two: January 16, 2025
  • Initial Release Candidate: February 6, 2025
  • Final Release Candidate: February 20, 2025
  • General Availability: March 18, 2025
Jakarta EE 11
  • TCK refactoring for Jakarta EE Core Profile complete
  • The Jakarta EE Platform project will initiate the release process once an implementation passes the Core Profile TCK and files a Compatibility Certification Request
  • Possible release date in mid- to late-November 2024
