What is the Jakarta EE Future Directions Group?
  • Intended to bring together people from inside and outside of Java related working groups, influencers, and other interested parties
  • Accepts the role of ideation, research, and presentation of recommendations, working closely with the working groups to ensure the ideas are scoped, prioritized, and understood.
  • Learn more at https://projects.eclipse.org/interest-groups/jakarta-ee-future-directions
Who are the Jakarta EE Ambassadors?
  • An independent grassroots group of people committed to moving Jakarta EE forward through active community participation and advocacy
  • Leadership Council
    • Ryan Cuprak (President)
    • Michael Redlich (Vice President)
  • Learn more at https://jakartaee-ambassadors.io

Mike's 30-Year Java Journey

  • 1995: Rutgers students downloading JDK
  • 1997: C++/Java debates at heritage-Exxon
  • 2001: Formed ACGNJ JUG
  • 2001: Added Java talks to TCF portfolio
  • 2005: Started collaboration with Barry Burd on Design Pattern technical articles on Java Boutique
  • 2016: Started writing for InfoQ
  • 2020: Rebranded ACGNJ JUG to Garden State JUG
  • 2021: Elevated to lead Java editor
  • 2022: Elected committer to Jakarta NoSQL specification
  • 2023: Named Java Champion
  • 2023: Elected committer to Jakarta Data specification
